# 🤖 Telegram Bot NSTU (Korotkov)

Telegram-бот с поддержкой погоды, математических операций и запоминанием пользователей.

## 📋 Возможности

- 🌤 **Погода** - реальные данные через OpenWeatherMap API
- 🧮 **Калькулятор** - сложение, вычитание, умножение, деление
- 👤 **Запоминание имен** - бот помнит, как вас зовут
- ⏰ **Время** - показывает текущее время
- 💬 **Диалоги** - приветствия, прощания, "как дела"
- 📊 **База данных SQLite** - хранение истории всех пользователей

## 🚀 Команды

| Запрос | Пример | Ответ |
|--------|--------|-------|
| Приветствие | `Привет` | Здравствуйте! |
| Имя | `Меня зовут Иван` | Приятно познакомиться, Иван! |
| Погода | `Погода в Москве` | Температура, ветер, влажность |
| Сложение | `2 + 3` | 2 + 3 = 5 |
| Вычитание | `5 - 2` | 5 - 2 = 3 |
| Умножение | `4 * 3` | 4 * 3 = 12 |
| Деление | `10 / 2` | 10 / 2 = 5 |
| Время | `Сколько время` | Сейчас 14:35 |
| Как дела | `Как дела` | У меня всё отлично! |
| Прощание | `Пока` | До свидания! |
